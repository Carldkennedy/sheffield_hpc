.. _seff:

seff
======

The ``seff`` script can be used to display status information about a user's historical 
or running jobs.

Documentation
-------------

Documentation on the usage of the command can be found by running:

.. code-block:: console

    $ seff -h
    Usage: seff [Options] <Jobid>
       Options:
       -h    Help menu
       -v    Version
       -d    Debug mode: display raw Slurm data
